# APS2020Tutorial
Tutorial on image analysis with deep /machine learning for APS-2020 meeting in Denver

How to use it:

Option 1:

1. Open a notebook (file with .ipynb extension)
2. Click on 'Open in Colab' icon at the top of each notebook
3. Run it in Google Colaboratory

Option 2:
1. Go to Colab [page](https://colab.research.google.com/github/ziatdinovmax/APS2020Tutorial) (it may ask for GitHub authorization).
2. It will show the list of available notebooks from this repository, which you can open and run in Google Colaboratory.
